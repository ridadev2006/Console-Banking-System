🏦 Console Banking Management System (Pre-OOP)

A console-based banking management system built in C++.

This project was developed in November last year during my early C++ learning phase, before studying Object-Oriented Programming (OOP).
It represents one of my first attempts at building a complete, structured application using procedural programming concepts.

📌 Project Overview

The system simulates a basic banking environment where users can manage clients and perform financial transactions.
All data is stored in text files and processed using structured logic.

✨ Features

Client Management (Add / Update / Delete / Find)

User Management System

Login & Authentication

Permission-Based Access Control (Bitwise Permissions)

Deposit & Withdraw Operations

Total Balance Calculation

Menu-driven console interface

🛠 Technical Concepts Practiced

struct for data modeling

enum for menu and permission control

vector for in-memory data handling

File handling using fstream

String parsing & custom split functions

Bitmask permission logic

Modular function design

This project is part of my learning journey and reflects my understanding of structured programming before transitioning to Object-Oriented Programming and more advanced software design practices.
